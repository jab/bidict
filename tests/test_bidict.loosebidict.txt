Test script for bidict.loosebidict::

    >>> from bidict import loosebidict

Initializing with different keys mapping to the same value succeeds::

    >>> loosebidict([(1, 1), (2, 1)])
    loosebidict({2: 1})

Initializing with collapsing mappings succeeds::

    >>> loosebidict([(1, 1), (2, 2), (1, 2)])
    loosebidict({1: 2})

Collapsing updates succeed::

    >>> b = loosebidict({1: 1, 2: 2})
    >>> b[1] = 2
    >>> b
    loosebidict({1: 2})
    >>> b = loosebidict({1: 1, 2: 2})
    >>> b.inv[2] = 1
    >>> b
    loosebidict({1: 2})
    >>> b = loosebidict({1: 1, 2: 2})
    >>> b.update({1: 2})
    >>> b
    loosebidict({1: 2})
